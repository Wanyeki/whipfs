# whipfs - nodejs api


## End points

### Create user
```bash
post: https://whipfs.web.app/user/register
```
```json
{ 
    "firstName":"joseph",
    "lastName":"mutinda",
    "email":"Joseph@gmail.com",
    "password":"password", 
    "phoneNumber":"254741685607"
}
```
